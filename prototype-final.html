PROTOTYPE FINAL

OVERVIEW

We have decided to make our application in react and will be creating components for that. I am responsible for login and signup pages and the front dashboard.

The front end as mentioned earlier, will be based on React and the back end will be base on PHP and MY Sql

FEATURES

  • Users can Register and Login if already registered
  • Upon registeration they will be given their separate database for storing the data.
  • User's information will be validated before the signup
  • In dashboard users can see the data related to their medicines
  • They can also perform the basic functions like sell medicines, add/delete medicines and search for the medicines

PROJECT USER STORIES

  • As a pharmacy manager i want to see my data in real-time.
  • As a customer, i want to get online receipts rather than printed one's. I care for environment
